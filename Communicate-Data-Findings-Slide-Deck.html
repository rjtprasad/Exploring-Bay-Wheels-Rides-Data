Patterns in Bay Wheels Dataset¶

by RAJAT PRASAD¶

The main area of focus:¶

  1. Proportion of rides done by each user type.
  2. Distribution of ride duration and distance.
  3. Average ride duration and distance for each user type.
  4. Rides by days of week and month for each user type.
  5. Peak hours of rides.
  6. Hourly and Weekly Rides for Each User Type.

Dataset Overview¶

In this project I am using Bay Wheels ride dataset for year 2019 from month January to December. The dataset after cleaning contains about 2262979 rows and 22 columns.

For original dataset and list of all variables avilable in dataset https://www.lyft.com/bikes/bay-wheels/system-data.

Proportion of Rides Done by Each User Type¶

About 82.23% of rides are done by subscriber of Bay Wheels bicycle sharing system and about 17.77% of rides are done by non subscriber.

Rides by Days of Week and Month for Each User Type¶

  • Rides done by customer is nearly equal to each days of weeks but subscriber rides more on regular days than weekends.
  • Subscriber ride more in March and April and very less in July.
  • Customer rides less wholes year compare to subscriber.
